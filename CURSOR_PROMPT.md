# Cursor Agent 提示词

请使用以下提示词启动开发：

---

**任务**：构建一个轨道交通导视生成器

**要求**：
1. 创建 web 应用
2. 实现画布系统，支持多个固定尺寸的导视牌（1200x400, 2400x800, 3600x1200 像素）
3. 创建元素库，包含：
   - 出入口标识（A/B/C/D + 箭头）
   - 线路名称组件
   - 图标：电梯、扶梯、无障碍、卫生间、便利店、ATM
4. 实现拖放功能，元素可以从侧边栏拖到画布
5. **关键**：实现智能吸附系统
   - 元素靠近时自动对齐（边缘/中心）
   - 显示辅助线
   - 等距分布检测
   - 吸附阈值：10 像素
6. 设计风格：现代扁平化，参考北京/东京/伦敦地铁设计
   - 高对比度配色
   - Sans-serif 字体
   - 圆角矩形按钮
   - 清晰的图标线条
7. 导出功能：PNG（300dpi）和 SVG
8. 使用 uv 管理依赖

**开始时**：
- 先创建基础窗口和画布
- 然后实现一个简单的矩形元素拖放
- 逐步添加吸附逻辑
- 最后丰富元素库和样式

请开始编写代码！
