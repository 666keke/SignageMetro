# 轨道交通导视生成器 (Metro Signage Generator)

## 项目概述
一个基于 **Python + uv** 的现代化**Web 轨道交通导视牌设计工具**。提供固定尺寸模板、可视化画布（WYSIWYG）、拖拽组件、智能吸附对齐，并支持 PNG/SVG 导出与项目文件保存。

---

## 核心功能

### 1. 固定尺寸的 Signage 板（Web 画布）
* 提供多种标准尺寸导视牌模板  
    * 常见尺寸：1200×400mm、2400×800mm、3600×1200mm 等
* 浏览器内可视化画布，所见即所得编辑
* 支持缩放、平移、标尺/参考线（可选）

### 2. 可拖动元素系统（组件库 + 画布拖拽）
支持的元素类型：
* **出入口标识**：A、B、C、D 出口，带方向箭头
* **线路名称**：线路号、站名、方向信息
* **提示标志**：
    * 电梯/扶梯
    * 无障碍设施
    * 卫生间
    * 便利店/ATM
    * 换乘标识
* **图标库**：通用交通图标集（SVG 为主，便于缩放与导出一致）

交互能力：
* 拖拽添加、移动
* 选择、框选、多选
* 缩放、旋转（可选，取决于规范）
* 属性面板：字体、字号、颜色、间距、图标变体等

### 3. 智能吸附与对齐（前端实时）
* 网格吸附 + 关闭/开启
* 元素边缘/中心吸附
* 辅助线（对齐线）显示
* 等距分布、一键对齐（左/右/上/下/水平居中/垂直居中）
* 智能间距建议（基于视觉规范的默认间距、留白规则）

### 4. 设计风格（主题系统）
* 现代扁平化设计语言
* 可配置主题（颜色、字体、线宽、圆角、图标风格）
* 融合各系统优点：
    * 北京/上海：清晰层级结构
    * 东京：高密度信息组织
    * 伦敦：经典线路标识元素
    * 新加坡：简洁色彩系统
* 高对比度、可读性优先（可选：色弱友好模式）

## 技术栈（Web 版）

### 后端（Python）
* **Python 3.11+**
* **uv**：包管理与运行
* **FastAPI**：API 服务（项目保存/加载、导出、资源管理）
* **Pydantic**：数据模型（画布/元素/样式/版本）
* **存储**：
    * 初期：本地文件（JSON + SVG/PNG）
    * 可选：SQLite / Postgres（多用户与历史版本）

### 前端（浏览器画布与交互）
推荐两种路线（二选一）：
* **SVG/DOM 方案**（更适合导视牌矢量与排版一致性）
    * React / Vue + SVG 编辑层
    * 优点：导出 SVG 更直接、文字排版更稳定
* **Canvas/WebGL 方案**（更适合大量元素与高性能）
    * Konva.js / PixiJS / Fabric.js
    * 优点：复杂交互性能更好

常用能力：
* 状态管理：Zustand/Pinia/Redux（任选）
* 拖拽/选择/变换：库能力 + 自研对齐吸附逻辑
* 字体加载：WebFont + 字体子集（可选）

### 导出与渲染
* **SVG 导出**：前端直接生成（推荐）或后端二次校验
* **PNG 导出**：
    * 前端：SVG 渲染到 Canvas 后导出
    * 后端：CairoSVG / Pillow / headless 浏览器渲染（需要更一致的出图可选）
* 可选增强（后端高质量排版/渲染）：
    * **Cairo / Pango**：高质量文字与矢量渲染

### 配置与项目格式
* **JSON**：工程文件（含画布尺寸、元素列表、样式、资源引用、版本号）
* 资源：内置图标库（SVG）+ 自定义上传（可选）

## 设计原则
1. **直观性**：浏览器打开即用，拖拽即可完成设计  
2. **精确性**：毫米级尺寸体系 + 像素级对齐与吸附  
3. **效率**：模板/主题/组件复用，快速标准化输出  
4. **美观**：现代扁平化 + 高对比可读性 + 跨系统一致性
